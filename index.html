소개 について ABOUT

코드와 픽셀로 물건을 만드는 디지털 장인의 작업실입니다.
키보드는 망치, 모니터는 작업대. 완벽한 것은 없지만, 모든 작업물에는 영혼이 담겨 있습니다.

コードとピクセルで物を作るデジタル職人の工房です。
キーボードはハンマー、モニターは作業台。完璧なものはないが、すべての作品に魂が込められています。

A digital craftsman's workshop where things are built with code and pixels.
Keyboard as hammer, monitor as workbench. Nothing is perfect, but every work has soul.

철학 哲学 PHILOSOPHY

우리는 속도보다 깊이를, 트렌드보다 본질을 추구합니다.
디지털 시대의 장인정신은 0과 1 사이 어딘가에 존재하며,
모든 버그는 배움의 기회, 모든 프로젝트는 실험입니다.

私たちは速度より深さを、トレンドより本質を追求します。
デジタル時代の職人魂は0と1の間のどこかに存在し、
すべてのバグは学びの機会、すべてのプロジェクトは実験です。

We pursue depth over speed, essence over trends.
Digital craftsmanship exists somewhere between zeros and ones,
Every bug is a learning opportunity, every project an experiment.

작업물 作品 WORKS

웹 개발 ウェブ開発 Web Development

정적 사이트부터 인터랙티브 앱까지, 웹의 모든 영역을 탐험합니다. 静的サイトからインタラクティブアプリまで、ウェブのすべての領域を探検します。 From static sites to interactive apps, exploring every corner of the web.

디자인 デザイン Design

기능과 미학의 균형, 사용자 경험에 대한 끊임없는 고민. 機能と美学のバランス、ユーザー体験への絶え間ない考察。 Balance of function and aesthetics, constant reflection on user experience.

실험 実験 Experiments

새로운 기술, 새로운 표현 방식. 실패해도 괜찮은 안전한 실험실. 新しい技術、新しい表現方法。失敗しても大丈夫な安全な実験室。 New tech, new expressions. A safe lab where failure is okay.
